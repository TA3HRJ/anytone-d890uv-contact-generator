How to Import

  1. Download the CSV file for your region
  2. Open AnyTone CPS software
  3. For DMR: go to DMR > Digital Contact List
  4. For NXDN: go to NX > NX Digital Contact List
  5. Use Tool > Import and select the CSV file
  6. Write the codeplug to the radio
